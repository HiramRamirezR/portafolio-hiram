Project Description

"Hilos" is a web application that allows users to transform their images into stunning "thread art" generated artworks. The application offers an intuitive interface to upload images, preview the result, and optionally purchase a physical kit to create the artwork in real life.

Features

  • Thread Image Generation: Converts any uploaded image into an artistic thread representation.
  • Real-time Preview: Allows users to see how their transformed image will look before finalizing.
  • Stripe Payment Integration: Facilitates a secure purchase process for acquiring the thread kit.
  • Email Confirmation: Sends an email to the user with a unique link to a personalized viewer of their thread image.
  • Custom Thread Viewer: A dedicated web application where users can view their generated thread image and associated data to replicate it.
  • Data Persistence: Uses a database to store user and purchase information.

Technologies Used

Backend

  • FastAPI: High-performance web framework for building APIs.
  • Python: Main programming language.
  • OpenCV (`opencv-python-headless`): For image processing and thread generation logic.
  • NumPy: For efficient numerical operations.
  • Pillow: Image processing library.
  • Uvicorn: ASGI server to run the FastAPI application.
  • SQLAlchemy: ORM for database interaction.
  • Pydantic: For data validation and configuration management.
  • `python-dotenv` : For environment variable management.
  • Stripe: For payment processing.
  • `smtplib` : For sending confirmation emails.

Frontend

  • HTML5: Web page structure.
  • CSS3: Styles and responsive design.
  • JavaScript: Client-side interactive logic, including image loading, progress simulation, and API integration.
  • Font Awesome: Icons.
