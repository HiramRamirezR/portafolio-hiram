Project Description

This project is a comprehensive management system designed for academies or schools, ideal for managing students, families, and payment tracking. Its main purpose is to simplify the administration of student information and their financial transactions, providing an efficient tool for control and decision-making.

Purpose and Benefits

  • Centralized Student Management: Allows registration, consultation, and editing of detailed information for each student, including personal data, grades, and entry/exam dates.
  • Family Administration: Facilitates grouping students by families, simplifying payment management and group fee application.
  • Efficient Payment Control: Records payments, edits and deletes transactions, and maintains a clear status of each family's payment history (current, overdue, or unpaid).
  • Financial Tracking: Includes functionalities to manage savings associated with families, offering a more complete view of finances.
  • Detailed Reports: Generates payment reports, aiding in accounting and financial analysis.
  • Intuitive Interface: Designed with a clean and user-friendly interface, reducing the learning curve and improving productivity.
  • Security and Authentication: Protects access to information through a user authentication system.
  • Cloud Database: Uses Firebase Realtime Database, ensuring data availability and real-time synchronization.

Technologies Used

Frontend

  • HTML5: For the structure and content of web pages.
  • CSS3: For the design and visual style of the application.
  • Bootstrap 4: CSS framework providing pre-designed components and a responsive grid system for a modern and adaptable interface.
  • JavaScript (ES6+): For client-side logic, interactivity, and DOM manipulation.

Backend/Database

  • Firebase: Google development platform used for:
    • Firebase Authentication: User management and secure authentication.
    • Firebase Realtime Database: NoSQL cloud database for storing and synchronizing data in real-time.

Additional JavaScript Libraries

  • SweetAlert2: For creating customized and attractive alerts and notifications.
