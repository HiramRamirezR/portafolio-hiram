Project Description

A classic implementation of the Hangman game where the player has to guess a hidden word, letter by letter, within a limited number of attempts. This project was developed to strengthen fundamental programming skills and demonstrate the ability to create interactive applications from scratch.

Features

  • Interactive Gameplay: Allows the user to input letters to guess the hidden word.
  • Visual Feedback: Displays the hangman's drawing progressively with each incorrect guess, along with the letters already used.
  • Random Word Selection: The game randomly chooses a word from a predefined list, ensuring a different experience in each session.
  • Clear and Simple Interface: The game runs in the console with a clean and easy-to-understand design.

Technologies Used

Backend

  • Python: Main programming language for the game's logic.
